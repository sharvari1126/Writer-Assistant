# Writer Assistant 🌙

Writer Assistant is a modular Python-based application designed for writers, poets, and creative thinkers.  
The project focuses on language assistance features such as meanings, synonyms, rhymes, and phrase-to-word suggestions, while being designed for future NLP and AI integration.

---

## Features

- Word Meaning Lookup
- Synonym Finder
- Rhyme Finder
- Phrase-to-One-Word Suggestions
- Modular API-based architecture
- Interactive command-line interface

---

## Technologies Used

- Python
- Requests Library
- Datamuse API
- DictionaryAPI
- Git & GitHub
